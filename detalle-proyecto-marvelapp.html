App de Superhéroes de Marvel

Este proyecto tiene como objetivo proporcionar una aplicación de muestra que muestra una lista de superhéroes de Marvel y su información detallada utilizando la API de Marvel. La aplicación está construida utilizando la arquitectura MVVM (Modelo-Vista-Modelo de Vista) y Jetpack Compose, una biblioteca moderna de IU de Android que simplifica el desarrollo de interfaces de usuario dinámicas y atractivas.

El proyecto está construido siguiendo Clean Architecture, lo que permite separar adecuadamente las responsabilidades y mejorar la escalabilidad del código. Para consumir la API de Marvel, se utiliza Retrofit, una biblioteca de cliente HTTP que simplifica el consumo de API. Además, se implementa un patrón de repositorio para gestionar los datos y separar la capa de datos de la capa de presentación.

La aplicación ofrece una experiencia de usuario fluida y atractiva al mostrar la información detallada de los superhéroes y permitir a los usuarios explorar el vasto universo de Marvel de manera sencilla y eficiente.

Equipo: David Antonio Gonzalez Blanchard
